## Project Description
This project is a **Mini Authentication System** that allows users to:

- Register an account
- Log in using valid credentials
- Access a protected dashboard/profile
- Log out securely

The system follows a **3-layer architecture**:
- Frontend: React
- Backend: Spring Boot
- Database: MySQL / PostgreSQL
